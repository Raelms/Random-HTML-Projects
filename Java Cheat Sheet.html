Java Cheat Sheet

1. Introduction to Java

Java is a high-level, class-based, object-oriented language that is platform-independent via the Java Virtual Machine (JVM).

  • JDK: Java Development Kit (tools for development)
  • JRE: Java Runtime Environment (to run Java apps)

2. Basic Objects

String s = "Hello";
Integer num = 42;
Double d = 3.14;
Boolean b = true;

Java wraps primitive types in classes like Integer and Boolean for use in collections and generics.

3. Basic Methods & Classes

class Dog {
    String name;
    Dog(String name) {
        this.name = name;
    }
    void bark() {
        System.out.println(name + " says Woof!");
    }
}

Classes define objects. Methods define object behavior.

4. Data Types

int x = 10;
long l = 100000L;
double pi = 3.1415;
char c = 'A';
boolean flag = false;

Java has 8 primitive types: byte, short, int, long, float, double, char, and boolean.

5. Branches

if (x > 0) {
    System.out.println("Positive");
} else {
    System.out.println("Non-positive");
}

switch (x) {
    case 1: System.out.println("One"); break;
    default: System.out.println("Other");
}

Use if or switch to perform conditional logic.

6. Loops

for (int i = 0; i < 5; i++) {
    System.out.println(i);
}

Other loops include while and do...while.

7. Arrays

int[] nums = {1, 2, 3, 4};
for (int n : nums) {
    System.out.println(n);
}

Arrays are fixed-size data structures.

8. Methods Continued

public int add(int a, int b) {
    return a + b;
}

Methods can return values and take parameters. Use static for class-level methods.

9. Classes Continued

class Cat {
    private String breed;
    public String getBreed() { return breed; }
    public void setBreed(String breed) { this.breed = breed; }
}

Encapsulation hides internal states using private and accessors.

10. Inheritance

class Animal {
    void eat() { System.out.println("Eats food"); }
}
class Bird extends Animal {
    void fly() { System.out.println("Flies"); }
}

Java supports single inheritance with extends.

11. Generics

class Box<T> {
    private T item;
    void set(T item) { this.item = item; }
    T get() { return item; }
}

Generics allow classes to operate on typed data.

12. Collections

List<String> list = new ArrayList<>();
Map<String, Integer> map = new HashMap<>();

Java collections include List, Set, and Map interfaces and their implementations.

Bonus: Obscure Java Features

// Anonymous Class
Runnable r = new Runnable() {
    public void run() {
        System.out.println("Run");
    }
};

// Varargs
public void printAll(String... args) {
    for (String s : args) {
        System.out.println(s);
    }
}

// Enum
enum Color { RED, GREEN, BLUE }

// Static Block
static {
    System.out.println("Runs once when class loads");
}

// Reflection
Method[] methods = String.class.getDeclaredMethods();

These features allow for more flexible and advanced programming patterns.
